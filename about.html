About SkillSwap

SkillSwap: Student Talent Exchange Platform is a database-driven web application created for the Business Professionals of America (BPA) Virtual Web Application Team event. Our goal was to design a secure, user-friendly platform that allows students to share their skills, request academic or creative help, and collaborate with peers in a structured environment.

The application supports user registration and login, profile creation, session requests, scheduling, and administrative oversight. SkillSwap was developed to promote peer learning, accountability, and community engagement among students.

BPA Event Overview

This project was developed as part of the BPA Web Application Team (V04) event. Teams of 2–4 contestants are tasked with building a fully functional web application using original code, a database-driven backend, and server-side functionality.

  • Database-driven user profiles and requests
  • Server-side logic and authentication
  • Admin tools for monitoring activity
  • Responsive, accessible front-end design
